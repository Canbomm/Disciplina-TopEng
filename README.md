# Disciplina-TopEng
Disciplina: Tópicos em Engenharia

Semestre UnB: 2023.2

- IAs simples.
- IAs DNN (Deep neural network).
- IAs CNN (Convolucionais).
- Como usar IAs Pré-Treinadas.
- IAs LSTM (Long Short Term Memory).
